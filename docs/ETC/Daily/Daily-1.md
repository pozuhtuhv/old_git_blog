## 고통의 고통이었다

방법을 알아가는데만 한 몇일이 걸렸다.
구조를 몰랐던게 너무 크다.

commit 과정에서 오류가 났고, 과정에서 오류가 안나는 방법을 찾는것에 시간을 거의 다 뺐긴 것 같다
